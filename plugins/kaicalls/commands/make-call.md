---

Place an outbound call with KaiCalls only after approval.

Before calling, confirm:

- Agent to use
- Recipient phone number in E.164 format
- Recipient name, if available
- Call purpose and context
- Whether this is one call or a batch

After approval, start the call and report the status, summary, and next steps.
