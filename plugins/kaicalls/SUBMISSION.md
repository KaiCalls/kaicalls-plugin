# Submission Readiness

This is the official KaiCalls plugin package.

## Public Submission Checklist

- `.claude-plugin/plugin.json` exists and validates with `claude plugin validate --strict`.
- `.codex-plugin/plugin.json` exists and validates with the Codex plugin validator.
- `.mcp.json` points to the hosted KaiCalls MCP connector.
- Skills are public-facing and do not mention private repos, private paths, internal client names, or operator-only workflows.
- README and setup docs explain OAuth/MCP and API-key alternatives.
- Examples use fake IDs, fake phone numbers, and placeholder tokens.
- License, support, and security files are present.

## GitHub-Link Submission

Repository URL:

```text
https://github.com/KaiCalls/kaicalls-plugin
```

## Zip Submission

The plugin can also be submitted as a zip containing this directory structure. No public repository URL is required for the zip path.
